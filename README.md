# Music Shop

An online music store build with React, this project is part of The Odin Project's curriculum: https://www.theodinproject.com/lessons/react-new-shopping-cart

[Live Demo](https://react-shop-5lc.pages.dev/)

# Features:

- Routing:
  - Two main routes: Home and Shop

  - Nested routes within Shop: Product details and Checkout

- Product filtering and sorting functionality

- Shopping cart management

- Product image carousel 

- Credit card number validation
